# API-Alunos

API REST desenvolvida em Python com FastAPI, criada para fins acadêmicos
com o objetivo de praticar operações de CRUD, modelagem de dados e validação
de informações utilizando Pydantic.

O projeto simula o gerenciamento básico de alunos, permitindo criar, listar,
atualizar e remover registros por meio de endpoints REST.

## Tecnologias utilizadas
- Python
- FastAPI
- SQLAlchemy
- Pydantic
- SQLite

## Observação
Este projeto possui finalidade exclusivamente acadêmica e de aprendizado.
